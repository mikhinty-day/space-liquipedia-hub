# space-liquipedia-hub

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或网站，仅作为通用 HTML 资源的集合与发布点。

## 项目简介

space-liquipedia-hub 是一个静态 HTML 页面归档仓库。我们收集、整理并托管多种独立的前端页面，涵盖信息展示、工具界面、文档视图等形式。每个页面均为自包含的 HTML 文件，不依赖外部服务或特定后端。

## 目录结构

```
.
├── pages/            # 存放所有独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/           # 页面共享的静态资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── index.html        # 页面索引或导航页（可选）
└── README.md         # 本文件
```

## 页面归档说明

- 每个页面存放于 `pages/` 目录下，文件命名应体现内容主题，使用小写字母和连字符。
- 页面应尽可能自给自足，所需资源优先引用 `assets/` 目录中的文件。
- 若页面需要额外说明，可在 `pages/` 目录下建立同名 `.md` 文档（如 `page-01.md`）。
- 归档内容不涉及任何具体域名、网站或商业推广信息。

## 维护说明

- 欢迎提交 pull request 添加新的独立 HTML 页面，或改进现有页面。
- 新增页面请确保没有外部依赖或敏感内容。
- 提交前请检查 HTML 结构是否完整、资源路径是否正确。
- 本仓库仅用于归档和发布，不提供内容审核之外的额外服务。
